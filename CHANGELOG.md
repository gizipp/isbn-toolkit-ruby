# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-28

### Added
- ISBN validation (ISBN-10 and ISBN-13)
- ISBN-10 ↔ ISBN-13 conversion
- EAN-13 support
- Formatted output with hyphens
- Open Library metadata lookup
- Google Books metadata lookup
- ActiveRecord validation (`validates :isbn, isbn: true`)
- Check digit calculation
